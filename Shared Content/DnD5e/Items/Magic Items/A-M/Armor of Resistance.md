## Description

You have [[resistance]] to one type of damage while you wear this [[2 - Armor|armor]]. The GM chooses the type or determines it randomly from the options below.

`dice: [[Armor of Resistance (DMG)]]^ArmorResist]]`

| **d10** | **Damage Type**                                       |
| ------- | ----------------------------------------------------- |
| 1       | [[Acid]]                                              |
| 2       | [[Cold]]                                              |
| 3       | [[Fire]]                                              |
| 4       | [[Force]]                                             |
| 5       | [[Lightning]]                                         |
| 6       | [[Necrotic]]                                          |
| 7       | [[0. DM Screen/Weapons, Armor, Items, Spells/Poison]] |
| 8       | [[Psychic]]                                           |
| 9       | [[Radiant]]                                           |
| 10      | [[Thunder]]                                           |
^ArmorResist
