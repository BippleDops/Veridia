# Dice Set

## Overview
**Price**: 1 sp

**Weight**: â€”

***Gaming Set.*** This item encompasses a widerange of game pieces, including dice and decks of cards (for games such as Three-Dragon Ante). A few common examples appear on the [[5 - Tools|Tools]] table, but other kinds of gaming sets exist. If you are proficient with a [[Gaming Set|gaming set]], you can add your [[3 - Proficiency Bonus|proficiency bonus]] to [[4 - Ability Checks|ability checks]] you make to play a game with that set. Each type of gaming set requires a separate proficiency.
