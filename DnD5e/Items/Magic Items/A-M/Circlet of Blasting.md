# Circlet of Blasting

## Overview
**Type**: Magic Item

**Group**: *Wondrous item*

**Rarity**: *Uncommon*

## Description
While wearing this circlet, you can use an action to cast the **[[Scorching Ray|scorching ray]]** spell with it. When you make the spell’s attacks, you do so with an [[14 - Actions in Combat|attack]] bonus of +5. The circlet can’t be used this way again until the next dawn.
