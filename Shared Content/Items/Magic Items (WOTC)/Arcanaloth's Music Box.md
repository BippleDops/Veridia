# Arcanaloth's Music Box

**Type:** Mundane

**Rarity:** unknown (magic)

**Cost:** 75000 cp

## Description

This music box is made of dark wood with gold filigree. Each of its five sides is sculpted with the image of a horned woman playing a different [[musical instrument]]: a [[2. Mechanics Reference Almanac/Items/Mundane Items (WOTC)/Dulcimer]], a [[flute]], a harp, a [[2. Mechanics Reference Almanac/Items/Mundane Items (WOTC)/Lyre]], and a [[2. Mechanics Reference Almanac/Items/Mundane Items (WOTC)/Viol]]. While touching the box, you can use an action to make it play music featuring one of the instruments shown, which can be heard up to 60 feet away. You can also use an action to stop the music.
