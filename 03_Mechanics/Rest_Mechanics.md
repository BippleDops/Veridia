# Rest Mechanics

## Short Rest (1 hour)
- Spend Hit Dice to heal
- Recover some features:
  - Warlock spell slots
  - Fighter Action Surge
  - Monk Ki points
  - Bardic Inspiration (5+)

## Long Rest (8 hours)
- Regain all HP
- Regain half Hit Dice (min 1)
- Regain all spell slots
- Reset all features
- Remove exhaustion (1 level)

## Interruption
- 1+ hour walking
- 1+ hour combat
- Must restart
