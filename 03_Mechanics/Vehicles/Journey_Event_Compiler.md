# Journey Event Compiler

Framework to produce 1,000+ varied travel events by combining depth/height/merger categories with weather, faction, horror, and discovery layers. Detailed tables live in [[06_GM_Resources/Travel/Travel_Event_Table|GM Travel Tables]].

## Categories
- Environment: Weather, pressure, currents, winds, merger pulses
- Faction: Patrols, pirates, blockades, couriers, admirals
- Horror: Whispers, ghosts, infestations, ship hunger
- Discovery: Ruins, resources, routes, prophecies
- Systems: Mechanical failures, crew conflicts, memory leaks

## Dice Model
- Roll 1d5 for Category → Roll 1d20 for Subtable → Compose with 1d10 Complication
- Increase intensity by Zone Modifier (see [[Vehicle_Core_Rules#2) Piloting & Adaptation Requirements|Zone Modifiers]])
