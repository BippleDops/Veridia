# Filter
## Overview
**Price**: Llinen screen (1 sp per square foot) or silken screen (1 gp per square foot).

**Weight**: -

### Filter (Pic)
![[yswqljol.pbs.png|Filter (Pic)]]
[[yswqljol.pbs.png|open outside]]

## Description
We offer sheets of gauze or coarse-screen straw mesh (1 sp per square foot) that can be cut to fit most any laboratory setup. For those occasions in which a tighter filter is needed, we offer linen screen (1 sp per square foot) or silken screen (1 gp per square foot).

## Additional Details
10 square feet of each type is needed for a complete laboratory
