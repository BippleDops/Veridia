# Glaur

**Type:** Magic Item

**Group:** Wondrous Items, Instrument

**Rarity:** none

## Description

Short, curved horns like a cornucopia. Played with valves, glaur sound like trumpets, while those without valves, known as gloon, have a more mournful sound.
