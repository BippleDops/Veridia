# Emerald Pen

**Type:** Magic Item

**Group:** Wondrous Items

**Rarity:** uncommon

## Description

This pen is tipped with an emerald nib and requires no ink to write. While holding this pen, you can cast [[Illusory Script (Ritual)|Illusory Script]] at will, requiring no material components.
