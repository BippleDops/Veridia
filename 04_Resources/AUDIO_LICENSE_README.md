## 📜 License Information

All audio files in this vault are intended for use with open-source tabletop role-playing games and are organized according to Creative Commons licensing principles.

#### CC0 (Public Domain)
Files in these directories are free for any use:
- `/04_Resources/Assets/Audio/Generated/` - Procedurally generated ambient sounds
- `/11_Media/OpenSource/` - Community-contributed sounds

#### CC BY 4.0 (Attribution Required)
Files requiring attribution to original creators:
- `/04_Resources/Assets/Audio/Curated/` - Curated ambience tracks
- `/11_Media/Licensed/` - Licensed content with attribution

#### CC BY-SA 4.0 (Share-Alike)
Modified content that must maintain same license:
- `/11_Media/Remixed/` - Remixed and edited tracks
- `/04_Resources/Assets/Audio/Ambient/` - Modified ambient loops
