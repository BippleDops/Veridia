# Ring of Jumping

## Overview
**Type**: Magic Item

**Group**: *Rings*

**Rarity**: *Uncommon*

**Attunement**: *requires attunement*


## Description
While wearing this ring, you can cast the **[[Jump|jump]]** spell from it as a [[12 - The Order of Combat|bonus action]] at will, but can target only yourself when you do so.
