# Tocken
## Overview
**Price**: 25 gp.

**Weight**: -

## Description
These carved wooden bells lie upon a crossbeam and are struck by a hammer (they have no clappers). Tocken thus create a very pleasant ringing tone that contains the dark subtleties of the wood itself. All sizes: 25 gp.
