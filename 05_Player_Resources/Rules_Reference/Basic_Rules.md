# Basic Rules Reference

## Core Mechanics
- **Ability Checks**: d20 + modifier
- **Advantage/Disadvantage**: Roll twice
- **Proficiency**: Add bonus when proficient

## Combat
- **Initiative**: d20 + DEX
- **Attack**: d20 + modifier vs AC
- **Damage**: Roll damage dice
- **Healing**: Restore HP
