# Buying a Magic Item; Magic Item Price
*Source: Xanathar's Guide to Everything p. 126* 

**Magic Item Price**

| Rarity | Asking Price* |
|--------|---------------|
| Common | `(1d6 + 1) × 10` gp |
| Uncommon | `1d6 × 100` gp |
| Rare | `2d10 × 1,000` gp |
| Very rare | `(1d4 + 1) × 10,000` gp |
| Legendary | `2d6 × 25,000` gp |
^magic-item-price

*Halved for a consumable item like a potion or scroll
