# Ambush
*Source: Player's Handbook (2024) p. 94*  

When you make a Dexterity ([Stealth](/03_Mechanics/CLI/skills.md#Stealth)) check or an [Initiative](/03_Mechanics/CLI/variant-rules/initiative-xphb.md) roll, you can expend one Superiority Die and add the die to the roll, unless you have the [Incapacitated](/03_Mechanics/CLI/conditions.md#Incapacitated) condition.
