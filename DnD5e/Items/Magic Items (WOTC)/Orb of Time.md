# Orb of Time

**Type:** Magic Item

**Group:** Wondrous Items

**Rarity:** common

**Weight**: 3 lbs.

## Description

While holding this orb, you can use an action to determine whether it is morning, afternoon, evening, or nighttime outside. This property functions only on the Material Plane.
