--- # Hitting Cover

*Source: Dungeon Master's Guide p. 272* When a ranged attack misses a target that has cover, you can use this optional rule to determine whether the cover was struck by the attack. First, determine whether the attack roll would have hit the protected target without the cover. If the attack roll falls within a range low enough to miss the target but high enough to strike the target if there had been no cover, the object used for cover is struck.
