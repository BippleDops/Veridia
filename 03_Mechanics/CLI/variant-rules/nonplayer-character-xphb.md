# Nonplayer Character
*Source: Player's Handbook (2024) p. 371. Available in the <span title='Systems Reference Document (5.2)'>SRD</span> and the Free Rules (2024)* 

A nonplayer character (NPC) is a [monster](/03_Mechanics/CLI/variant-rules/monster-xphb.md) that has a personal name and a distinct personality.
