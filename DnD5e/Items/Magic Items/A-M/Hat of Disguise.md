# Hat of Disguise

## Overview
**Type**: Magic Item

**Group**: *Wondrous item*

**Rarity**: *uncommon*

**Attunement**: *requires attunement*

## Description
While wearing this hat, you can use an action to cast the **[[Disguise Self|disguise self]]** spell from it at will. The spell ends if the hat is removed.
