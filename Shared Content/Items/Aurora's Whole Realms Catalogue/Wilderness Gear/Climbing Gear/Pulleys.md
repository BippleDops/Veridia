# Pulleys
## Overview
**Price**: 15 gp each.

**Weight**: -

### Pulleys (Pic)
![[axdm0vhv.y1q.png|Pulleys (Pic)]]
[[axdm0vhv.y1q.png|open outside]]

## Description
A pulley will help lift heavy [[2. Mechanics Reference Almanac/Items/Items|items]] with rope by increasing leverage, and will also let [[2. Mechanics Reference Almanac/Items/Items|items]] roll across horizontally tied ropes (across a chasm, for example). 15 gp each.

## Additional Details
any adventurer with Mountaineering proficiency should have these [[2. Mechanics Reference Almanac/Items/Items|items]] to use his proficiency; DMs not using proficiencies may assign a +10% to climbing success rates (DMG Table 65)
