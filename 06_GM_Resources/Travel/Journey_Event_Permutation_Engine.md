*Dynamic travel encounters for every journey type*

## Core Event Matrix

### Base Event Types
Roll 2d10 and combine:

**First Die (Category)**:
1. **Environmental** - Weather, terrain
2. **Social** - NPCs, settlements  
3. **Combat** - Hostile encounters
4. **Discovery** - Ruins, treasures
5. **Mechanical** - Vehicle issues
6. **Political** - Faction encounters
7. **Mystical** - Magic phenomena
8. **Economic** - Trade opportunities
9. **Personal** - Character moments
10. **Weird** - Reality distortions

**Second Die (Intensity)**:
1-2: **Trivial** - Minor inconvenience
3-4: **Simple** - Quick resolution
5-6: **Moderate** - Requires effort
7-8: **Complex** - Multiple solutions
9-10: **Critical** - Major consequences

## Journey Pacing Framework

### Act 1: Departure (Choose 1)
- **Smooth Start** - Bonus to first day
- **Rough Launch** - Immediate complication
- **Ominous Signs** - Foreshadowing
- **Rushed Exit** - Forgot something

### Act 2: Rising Action (Roll 1d3 events)
- Space events across travel time
- Escalate intensity gradually
- Connect to character backstories
- Build toward climax

### Act 3: Climax (Choose or Roll)
- Major event from tables
- Combines previous elements
- Tests crew teamwork
- Significant consequences

### Act 4: Resolution (Choose 1)
- **Triumphant** - Bonus on arrival
- **Limping In** - Need repairs
- **Changed** - Permanent effect
- **Redirected** - New destination

## Multi-Day Journey Tracker

### Day Types (Roll each morning)
1-2: **Calm** - No events, full progress
3-4: **Active** - One minor event
5: **Eventful** - Two events
6: **Chaotic** - Major event

### Progress Modifiers
- **Favorable Conditions**: +25% distance
- **Normal Conditions**: Standard distance
- **Poor Conditions**: -25% distance  
- **Severe Conditions**: -50% distance
- **Impossible**: No progress

### Resource Depletion
Per day of travel:
- Food: 1 ration per person
- Water: 2 gallons per person
- Fuel: Varies by vehicle
- Morale: -1 if bad event
- Health: Track exhaustion

## Quick Event Generator

### Formula
1. Roll **Category** (d10)
2. Roll **Intensity** (d10)
3. Roll **Complication** (d6)
4. Optional: Add **Twist** (d8)

### Example Results
- **3 + 7 + 2**: Combat (Complex) + Time Pressure
  - "Sky pirates attack during storm"
  
- **4 + 2 + 5**: Discovery (Simple) + Hidden Agenda
  - "Ruins hide smuggler cache"
  
- **7 + 9 + 1**: Mystical (Critical) + Another Group
  - "Reality tear, rivals also affected"
